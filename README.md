## :checkered_flag: Starting ##

- Clone this project

```bash
git clone https://github.com/devenes/todos-api
```

- Go to the project directory
```bash
cd todos-api
```

- Run the following command to start the server:
```bash
go run main.go
```

- The server will initialize in the `http://localhost:8080` URL.
